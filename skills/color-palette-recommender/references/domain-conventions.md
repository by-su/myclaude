# Domain Conventions

각 분야에는 사용자가 "이건 그 카테고리답다"고 무의식적으로 느끼는 색 코드가 있다. 무조건 따라야 하는 룰은 아니지만, 의식하지 않은 채로 어기면 카테고리가 안 읽힌다. 의식적으로 어기면 강한 차별화 무기가 된다.

## Fintech / 자산 / 결제

**관습 색.** 네이비, 딥 블루, 다크 그린. 이유: 은행 200년 역사 + 안정감/돈/자연의 무의식적 코드.
**참고 브랜드.** Chase·Citi(네이비), 토스(네이비→블루), 카카오뱅크(옐로 액센트는 메신저 DNA), 신한SOL(짙은 블루).
**자주 보는 신호색.** 상승(좋음): 짙은 그린 (#0F9D58 류). **한국 주식/암호화폐는 빨강=상승, 파랑=하락** (글로벌과 정반대).
**피해야 할 것.** 너무 채도 높은 핫핑크/마젠타/네온 — 아무리 MZ 타깃이라도 "내 돈"을 맡기는 맥락에선 가벼워 보인다. "모던하고 트렌디"가 사용자 요청에 있더라도 Primary는 신뢰 톤으로, Accent에서만 트렌드를 풀어주는 게 안전하다.
**차별화 카드 (강함→약함 순서).**
- **차콜 + 골드/앰버** — Bloomberg Terminal × Apple Dark 톤. "디지털 골드" 내러티브에 강력. **암호화폐/투자 자산** 분야에서 가장 영리한 차별화 카드. 비트코인 자체가 골드 메타포라서 즉시 읽힌다.
- **보라/바이올렛** — Revolut, Linear, Arc 계열. 똑똑하고 모던한 "테크 자산". MZ 타깃 핀테크에서 가장 흔한 차별화.
- **차콜 + 민트/그린 액센트** — Cash App, 토스증권의 다크모드. 청량하고 부담 없는 입문 톤.
- **따뜻한 베이지/우드** — 국내 자산관리 신생 브랜드(WealthX, 박곰핀 등). "따뜻한 자산 상담사" 톤.

**암호화폐/Web3 특수 사항.**
- 한국 거래소(업비트·빗썸): 호가창에서 **빨강=상승, 파랑=하락**. brand primary가 빨강/파랑 계열이면 신호색과 충돌 → 사용자 인지 부조화. **brand primary와 시그널 컬러는 절대 같은 색역으로 쓰지 마세요.** 예: Primary가 mint면 상승은 mint가 아니라 빨강(`#F04452`)으로 유지.
- 글로벌 사용자 비중이 크다면 **빨강=하락, 그린=상승** 옵션 토글을 제공하는 게 안전.
- "디지털 골드" 앵글이 강력 — 비트코인을 가치 저장 수단으로 포지셔닝하면 골드 톤이 즉시 의미를 전달한다.

**Trust vs Trendy 텐션 해소법.** 사용자 요청에 "신뢰감"과 "모던/트렌디"가 함께 들어있으면 **Primary는 신뢰, Accent는 트렌드** 분리가 가장 안정적. Primary를 가장 모험적으로 갈수록 사용자가 "이거 진짜 내 돈 맡길 수 있나"는 의구심이 커진다.

## Healthcare / Wellness / 명상

**관습 색.** 부드러운 블루, 민트, 세이지 그린, 베이지. 이유: 청결/회복/자연.
**참고 브랜드.** Calm(딥 블루→그라데이션), Headspace(주황·옐로 = 의도적 차별화), Whoop(블랙+그린 = 테크-웰니스), Oura(차콜+골드 = 프리미엄), 마보(따뜻한 베이지).
**피해야 할 것.** 너무 채도 높은 빨강(병원 응급 코드 연상), 형광 컬러, 새까만 다크모드(병자가 죽음을 연상). 검정 일색 헬스케어는 거의 없다. **종교·뉴에이지 코드(보라+골드 = 영적, 인디고+퍼플 = 신비주의)** 도 피하라는 요청이 많다.
**차별화 카드.**
- 따뜻한 살구색/피치 (불안 완화 톤, Headspace가 대표적)
- 라벤더/페일 퍼플 (수면 앱)
- 모스/세이지 그린 (자연 회복)
- 차콜 + 골드 (Oura 톤, 프리미엄 테크-웰니스)
- 따뜻한 베이지+테라코타 (디지털 디톡스, 인간적 톤)

**의료기기/임상** 쪽은 조금 더 차갑고 중성적인 톤 (시멘트 그레이, 차가운 블루)이 신뢰감을 준다.

## Fitness / Sports / 운동

**관습 색.** 웜 에너지 hue(레드·오렌지·볼트 라임) + 블랙. 이유: 신체 활동·심박·땀·근력의 시각 번역. 운동 브랜드의 8할이 이 코드를 따른다.
**참고 브랜드.** Strava(오렌지 `#FC4C02` = 지구력·러닝), Apple Fitness+(무브 링 `#FA114F` 빨강 / 엑서사이즈 `#92E82A` 그린 / 스탠드 `#1EEAEF` 시안 — 3링 에너지 사이클), Nike Training Club(Volt `#CCFF00` + 블랙), Whoop(블랙 + 일렉트릭 그린 — 회복 데이터), Peloton(블랙 + 라이드 레드), Garmin(블랙 + 딥 블루 — 장비·내구성), MyFitnessPal(블루 = 데이터·기록).
**피해야 할 것.** 파스텔 톤 위주 (운동 강도 신호가 안 산다), 세이지/페일 그린 위주 (명상·스파 톤으로 미끄러짐 — Headspace·Calm 영역), 따뜻한 살구색 (헬스케어 wellness 톤). **헬스케어와의 분리가 가장 중요한 차별화.** "피트니스"라고 입력받았을 때 모델은 본능적으로 헬스케어/wellness 인접 코드를 빌려오는 경향이 있는데, 그러면 결과가 "스파 앱" 같아진다.
**차별화 카드.**
- 퍼포먼스·러닝·기록 → **블랙 + 오렌지/레드** (Strava·Peloton·Apple Move)
- 근력·코어 트레이닝 → **블랙 + Volt 라임/일렉트릭 옐로** (Nike Training Club)
- 회복·HRV·수면 통합 → **블랙 + 일렉트릭 그린** (Whoop) — 명상 앱의 세이지 그린이 아니라 *발광하는* 그린
- 장비·아웃도어·내구성 → **블랙 + 딥 블루/네이비** (Garmin·Suunto)
- 친근한 홈트·습관 빌딩·여성 타깃 → **따뜻한 코랄/피치 + 크림** (Sweat by Kayla Itsines, Tone It Up, Alo Yoga, Bandier) — wellness와 가장 가까운 영역이지만 *채도 높은 코랄*로 "땀 + 보상감"을 유지한다. 주의: Centr·Fitbod·Apple Fitness+ Light는 사실 이 톤이 아님(Centr=다크+레드, Fitbod=블루+화이트). "친근한 피트니스"라는 이름에 끌려서 그 브랜드를 빌려오면 안 된다 — 실제 이 톤은 여성 타깃 액티브웨어/피트니스 인접 라이프스타일 브랜드에서 온다.
- 게이미피케이션·MZ 피트니스 → **블랙 + 네온 컬러풀** (Zwift·Fitness+)

**운영 노트.**
- 운동 강도 게이지(저강도→고강도) 시그널 컬러가 brand primary와 충돌하지 않게 분리. Whoop이 회복 점수에 그린→옐로→레드 그라데이션을 쓰는 게 대표적.
- "Move·Exercise·Stand" 식의 멀티 메트릭이 있으면 Apple Fitness 3링처럼 Primary 외 별도 시그널 컬러 셋이 필요할 수 있음 — 한 색으로 다 처리하지 말 것.
- PR(Personal Record)/스트릭/달성 같은 "보상 모먼트" 액센트는 골드/앰버가 거의 항상 잘 먹힌다. 메인 brand color가 레드/오렌지 계열이면 이미 따뜻한 가족이라 자연스럽게 어울림.

**헬스케어/Wellness와의 명확한 분리.** "피트니스"는 활동·에너지·기록 코드, "Wellness/명상"은 회복·진정·자연 코드. 사용자가 "운동 + 명상" 같은 하이브리드를 요구하면 둘 중 어느 쪽이 메인 약속인지 명시적으로 묻고 그에 맞춰 가야 한다 — 한 팔레트에 두 코드를 동시에 강하게 섞으면 어수선해진다.

## F&B / 배달 / 푸드

**관습 색.** 따뜻한 주황, 빨강, 옐로. 이유: 식욕 자극. 맥도날드/버거킹/우버이츠/요기요 다 따뜻한 색.
**참고 브랜드.** 배달의민족(민트→옐로), 쿠팡이츠(핑크), 마켓컬리(보라 = 의도적 차별화로 "프리미엄 새벽배송"), 스윗밸런스(헬시 그린+크림), 풀무원(딥 그린+자연색).
**피해야 할 것.** 파랑·보라 위주 (식욕 감소, 자연계에 거의 없는 음식 색). 단, **컬리처럼 의도적으로 프리미엄/큐레이션 차별화**를 노릴 땐 보라가 강력해진다.
**차별화 카드.**
- 헬시푸드/샐러드 → **그린+크림+우드** (Sweetgreen, 스윗밸런스, dimi)
- 프리미엄 다이닝 → **차콜+골드+와인** (와인 셀러 톤)
- 디저트/베이커리 → **파스텔 핑크/크림+초콜릿 브라운**
- 한식/토속 → **테라코타, 머스타드, 옻갈/감색, 한지 크림** (적당히 톤 다운된 흙·식물·토기 색)
- 프리미엄 큐레이션 → **딥 퍼플/와인+크림** (마켓컬리 패턴)

**한식 + 식단관리**는 보통 두 톤이 충돌한다 — 한식 토속(따뜻한 갈색)과 헬시(그린)는 색조가 가깝지 않아서 한 팔레트 안에서 둘 다 강하게 쓰면 어수선해진다. 한 팔레트 안에서는 한쪽을 기준으로 잡고 다른 쪽은 액센트나 보조로 빼는 게 깔끔하다.

## Diet tracking / 식단관리 / Nutrition logging

**관습 색.** 헬시 그린 + 크림, 따뜻한 코랄/피치 + 크림, 데이터 블루 + 화이트. 이유: "음식 자체"가 아니라 "내 몸·습관·기록"이 주제라서 식욕 자극 톤(빨강/주황 강세)에서 한 발짝 떨어진다. F&B(배달·주문)와는 다른 색 코드.
**참고 브랜드.** Lifesum(`#7BC360` 헬시 그린 + 화이트 = 북유럽 클린 푸드), Noom(`#ED7950` 따뜻한 오렌지 + 크림 = 심리 기반 다이어트 코칭), MyFitnessPal(`#0066EE` 블루 = 데이터·로그·바코드 스캔), Yazio(헬시 그린 + 화이트 = 단식·간헐적 단식 트래커), Cronometer(차콜 + 라임 = 진지한 마이크로뉴트리언트 트래커), 다노(DANO, 따뜻한 코랄 + 크림 + 핸드라이팅 = 한국 여성 타깃 다이어트 코칭), 인밸런스(그린 + 우드 = 한국 헬시 식단 큐레이션), 핏블리(차콜 + 라임/오렌지 = 한국 진지한 운동·식단).
**피해야 할 것.** F&B 배달 톤의 채도 끝까지 끌어올린 빨강·주황 — 다이어트 맥락에선 "식욕 자극"이 오히려 역효과. 의료기기 톤의 차가운 화이트 + 차가운 블루 — 병원·진료 톤으로 미끄러진다. 강한 와인/보라 — 럭셔리 F&B 톤이라 일상 트래커 느낌이 죽는다. 너무 비비드한 핫핑크 — 다이어트 = "벌"로 읽힌다.
**차별화 카드 (강함→약함 순서).**
- **헬시 그린 + 크림 + 우드** — Lifesum·Yazio·스윗밸런스·인밸런스 코드. "내가 좋은 걸 먹고 있다"는 긍정 신호. 가장 안전하고 직관적인 헬시 트래커 톤. 채도 너무 높이면 Tailwind 그린 디폴트 느낌이 되니까 살짝 톤 다운(채도 40~55%).
- **따뜻한 코랄/피치 + 크림** — Noom·다노 코드. **다이어트를 "벌"이 아니라 "코칭/응원"으로 프레이밍**할 때 강력. 여성 타깃, 심리·습관 기반, 단식보다는 "꾸준함"에 가까운 서비스에 잘 맞는다. 코랄은 Tailwind red-400(`#F87171`)·rose-400(`#FB7185`)과 명확히 다른 톤(채도 낮추고 살짝 더 따뜻하게)으로.
- **차콜 + 라임/일렉트릭 그린** — Cronometer·핏블리·Whoop-adjacent 코드. **데이터 마니아·헬스 트레이너·체중 감량 진지파**용. 단순 헬시가 아니라 "마이크로뉴트리언트·매크로 정밀 트래킹" 포지셔닝일 때. 라임 그린은 발광감으로 데이터 강조를 만든다.
- **데이터 블루 + 차트 액센트** — MyFitnessPal·LoseIt 코드. 칼로리·매크로 차트·바코드 스캔 중심의 도구적 톤. 감정보다 숫자가 주인공인 트래커에 잘 맞지만 정서적 매력은 약하다.
- **소프트 파스텔 멀티(라벤더 + 민트 + 피치)** — Fastic·Simple 톤. 게이미피케이션·간헐적 단식·MZ 라이트 유저용. 친근하지만 진지한 다이어터에겐 약하게 보일 수 있다.

**운영 노트.**
- **목표·진척률 시그널 컬러 분리.** "목표 달성/초과/미달"을 brand color로 처리하면 사용자가 "내 점수가 brand color에 묶여있다"고 느껴 피로해진다. 진척률은 그린/옐로/레드 그라데이션을 brand color와 별도 색역에서 확보 (예: Primary가 코랄이면 진척률 그린은 brand과 무관한 #34A86B 류).
- **칼로리 초과 알림 색.** 빨강 직접 사용은 "벌"로 읽히는 경향. 톤 다운된 코랄/머스타드 정도로 부드럽게 가는 게 사용자 정서에 낫다 — 의료 응급이 아니라 일상 습관이라.
- **음식 사진과의 공존.** 식단 사진(빨강·주황·노랑·갈색 식재료)이 화면 절반 이상 차지하는 서비스라면, brand color는 사진과 hue가 겹치지 않게 그린·블루·차콜 계열이 안전. Primary를 오렌지로 두면 음식 사진 위에서 brand가 묻힌다.
- **여성 타깃 비중이 70%+면** 따뜻한 코랄/피치 톤 채택률이 압도적으로 높다 (Noom·다노·Sweat). 남성·진지파 비중이 높으면 차콜+라임 또는 데이터 블루로.
- **간헐적 단식(IF) 서비스**는 "공복/단식 윈도우"라는 시간 메타포가 강해서 다크 모드 또는 보라/네이비 톤이 의외로 강하게 작동한다 (Zero 앱이 대표적: 다크 + 보라).

**F&B 배달과의 분리.** F&B는 "음식 자체"가 주인공이라 식욕 자극(빨강·주황·옐로)이 메인이지만, 식단관리는 "내 몸·기록·습관"이 주인공이라 한 발 물러난 톤이 맞다. 사용자가 "맛집·배달"을 강하게 부각하면 F&B 카드로, "기록·다이어트·코칭"을 부각하면 이 카드로 분기.

**Fitness와의 분리.** Fitness가 "활동·에너지·기록(운동량)" 코드라면, 식단관리는 "선택·습관·자기 통제" 코드. 그래서 같은 그린이라도 fitness의 Volt 라임(`#CCFF00`)과 식단관리의 헬시 그린(`#7BC360`)은 정서가 다르다. 두 서비스를 합친 하이브리드라면 한쪽을 메인 약속으로 정하고 그 톤을 따른다.

## SaaS / B2B 협업 / 생산성

**관습 색.** 블루(노션 비포어, 슬랙 보라, 리니어 보라, 피그마 검정 등 사실 다양). 핵심은 "이성적이고 방해되지 않는" 톤.
**피해야 할 것.** 채도 너무 높은 풀컬러 — 작업하면서 8시간 보면 눈 아프다.
**차별화 카드.** 보라(노션/리니어 계열의 똑똑함), 그린(긍정/생산성), 다크모드 우선 디자인.
**B2B는 의외로 무난한 게 정답일 때가 많다.** 결정권자가 IT팀장이고 그 사람의 미감보다는 "사내에서 거부감 없는지"가 더 중요해서.

## 교육 / EdTech / 키즈

**관습 색.** 밝은 블루, 그린, 따뜻한 옐로. 이유: 신뢰 + 성장 + 활기.
**키즈**는 더 비비드하게 — 빨강/파랑/노랑 원색 + 일러스트. 단 학부모도 봐야 하니 너무 유치하면 거부감.
**차별화 카드.** 코딩 교육은 모노톤+컬러풀 액센트 (개발자스러운 분위기). 입시는 차분한 네이비/딥 그린 (진중함). 어학은 빨강/주황 (활동성).

## 게임 / 엔터테인먼트

**관습 색.** 다크 베이스 + 비비드 네온 액센트. 이유: OLED 화면 발광감, 몰입.
**차별화 카드.** 캐주얼 모바일 게임은 오히려 밝은 베이스 (애니팡, 캔디크러시). 코어 게이밍은 다크 + 보라/시안/레드 네온.

## 뷰티 테크 / AI Beauty / 퍼스널 컬러

**관습 색.** 로제/더스티 핑크, 소프트 코랄, 누드 베이지, 라벤더. 이유: 피부·입술·볼의 자연스러운 메이크업 색조를 서비스 UI에 반영하는 관습. 럭셔리 뷰티(블랙+골드)와 달리 "내 피부에 맞는 색을 찾는" 경험이 핵심이라 피부 톤에 가까운 따뜻한 중채도가 지배적.
**참고 브랜드.** Glossier(밀레니얼 핑크 `#F5C6C6` + 화이트 = 클린 뷰티 아이콘), 화해(Hwahae, 민트 그린 + 화이트 = 성분 분석 신뢰감), 글로우픽(코랄 + 화이트 = 뷰티 커뮤니티), Perfect Corp/YouCam(그라데이션 퍼플→핑크 = AI 가상 메이크업), Laka(라벤더+누드 = 젠더리스 뷰티), 헤라(Hera, 골드+블랙 = 프리미엄 K-뷰티), Charlotte Tilbury(앰버 골드 = 골든아워 글로우).
**피해야 할 것.** 순검정+골드 풀 럭셔리 톤(고급 화장품 매장 톤이라 "AI 테크 서비스"로 안 읽힘). 채도 높은 네온(메이크업 결과 이미지와 충돌 — 사용자 셀카/AI 이미지가 화면 주인공이어야 하므로 UI 색은 한 발 물러나야). 차가운 블루 위주(의료 진단·병원 톤으로 미끄러짐). 너무 비비드한 핫핑크/마젠타(10대 뷰티 놀이 톤으로 읽혀 유료 전환 신뢰 감소).
**차별화 카드 (강함→약함 순서).**
- **더스티 로즈 + 크림** — Glossier·에뛰드 코드. "내 피부에 녹아드는" 자연스러운 클린 뷰티 톤. K-뷰티 얼리어답터에게 가장 직관적. 가장 안전한 카드.
- **딥 플럼/모브 + 골드 액센트** — 마켓컬리식 보라 차별화를 뷰티에 적용. "AI가 분석해주는 프리미엄 서비스" 포지셔닝. 퍼스널 컬러 진단 = 전문성 코드.
- **웜 앰버/골든 + 크림** — Charlotte Tilbury·헤라 코드. "골든아워 글로우" 감성. 30대 여성·웨딩 준비 타깃에 강함. 럭셔리에 가장 가깝지만 블랙 없이 따뜻하게 풀어서 접근성 유지.
- **민트/세이지 + 화이트** — 화해·올리브영 코드. 성분 분석·피부 과학 이미지. AI 분석의 "정확도/신뢰" 쪽을 강조할 때. 단, 뷰티보다 헬스케어로 읽힐 위험.
- **라벤더 + 누드** — Laka·젠더리스 뷰티 코드. 남녀 모두 포용하는 톤. 남성 그루밍까지 아우르는 서비스라면 고려.

**운영 노트.**
- **셀카/AI 이미지와의 공존이 핵심.** 결과 페이지에서 사용자 셀카와 AI 생성 이미지가 화면의 절반 이상을 차지한다. UI 컬러가 피부색과 hue가 겹치면(빨강·주황·살구) 이미지 위에서 UI가 묻히거나 충돌. Primary는 피부색과 약간 떨어진 hue(로즈 340~355°, 플럼 270~290°, 앰버 28~35°)가 안전.
- **퍼스널 컬러 팔레트와의 분리.** 서비스가 "당신은 여름 쿨톤" 같은 컬러 팔레트를 보여주므로, UI 브랜드 컬러가 특정 퍼스널 컬러와 동일하면 사용자가 "이건 내 컬러야, UI야?" 혼동. UI는 중립적 톤으로 한 발 물러나는 게 안전.
- **Before/After 비교 UI.** 슬라이더 핸들, 프레임 컬러는 이미지 위에 떠야 하므로 이미지와 대비되는 색 필요 — 보통 순백 or 브랜드 Primary.

**Luxury 뷰티와의 분리.** 럭셔리 뷰티(샤넬·디올·맥)는 "매장 경험·프레스티지"가 핵심이라 블랙+골드+와인이 지배적. 뷰티 테크는 "AI 분석·맞춤 추천·일상 가이드"가 핵심이라 더 밝고 접근성 있는 톤이 맞다. 블랙 베이스를 쓰면 "화장품 브랜드 사이트"로 읽히고, AI 테크 서비스로 안 읽힌다.

## Luxury / 패션 / 뷰티

**관습 색.** 블랙, 화이트, 골드, 카멜, 와인. 이유: 비싼 광고지·잡지 80년의 유산.
**피해야 할 것.** 광택 있는 비비드 컬러, 채도 높은 형광색.
**차별화 카드.** 화이트+소프트핑크(클린 뷰티), 차콜+세이지(미니멀 럭셔리), 풀 컬러풀 그라데이션(MZ 패션 브랜드).

## Mobility / 모빌리티 / 자동차

**관습 색.** 블랙, 다크 그레이, 다크 블루 (자동차 자체가 거의 다 무채색이라).
**차별화 카드.** EV/그린 모빌리티는 그린+화이트. 차량 공유는 따뜻한 톤(우버블랙 vs 우버X 의도된 분리).

## Chat / Messaging / Community (실시간 채팅·메신저·커뮤니티)

**관습 색.** 보라/바이올렛(메신저 친근감 + 커뮤니티), 시안/하늘색(가벼움·신속), 그린(친근 메신저 — Line/WhatsApp), 옐로(친근 — 카카오/스냅챗). 가장 강한 단일 시각 코드는 **보라** (Discord·Slack·Twitch가 동시에 점령). 이유: 즉시성·친근감·휴대성·"내 화면 속 친구"의 따뜻한 디지털 톤.
**참고 브랜드.** Discord(블러플 `#5865F2` = 게이밍·커뮤니티), Slack(보라 `#4A154B` + 옐로/그린/마젠타 색종이 = 프로 메신저), Telegram(`#0088CC` 시안 = 가볍고 빠른 메신저), KakaoTalk(`#FAE100` 옐로 = 친근 국민 메신저), Line(`#06C755` 그린 = 친근 메신저), WhatsApp(`#25D366` 그린), Twitch(`#9146FF` 보라 = 게이머 채팅), Snapchat(`#FFFC00` 옐로 = Z세대), Reddit(`#FF4500` 오렌지 = 익명 커뮤니티), Yubo(보라/마젠타 = Z세대 랜덤 친구 만들기).
**피해야 할 것.** 너무 차분한 헬스케어·명상 톤(세이지·페일 블루) — 활기 없음. 너무 진지한 핀테크 네이비 — 메신저 친근감이 죽는다. 채도 한도까지 끌어올린 게임 풀 네온 — 어수선해진다. **메신저 그린(`#25D366`·`#06C755`)을 그대로 베끼면** WhatsApp/Line 카피로 즉시 읽힘. **빨강 Primary**는 신고/차단 시그널과 충돌해서 어색.
**차별화 카드 (강함→약함 순서).**
- **보라/바이올렛 + 라이트** — Discord·Slack·Twitch 코드. 가장 안전하고 직관적인 "온라인 커뮤니티" 코드. 게이밍·취미·랜덤 채팅 어디에나 잘 맞는다.
- **다크 베이스 + 네온 액센트(보라·마젠타·시안·라임)** — Discord 다크·Twitch 톤. 야간·게이밍 인접·MZ 커뮤니티에 강한 발광감.
- **시안/하늘색 라이트** — Telegram 코드. 가볍고 빠른 메신저, 진지하지 않은 분위기. 글로벌 톤.
- **소프트 파스텔(라벤더 + 피치 + 민트)** — Yubo·Honk 등 Z세대 친근 톤. 가벼움 + 안전감 + 따뜻함을 동시에. 익명 랜덤 채팅의 도발성을 누그러뜨리는 데 효과적.
- **옐로 + 친근 일러스트** — 카카오톡·스냅챗 코드. 너무 무거우면 안 되는 캐주얼 메신저.
- **다크 + 마젠타/핫핑크** — 익명·심야·랜덤 채팅의 살짝 도발적 톤. 데이팅(Tinder)으로 미끄러지지 않도록 주의: 데이팅은 보통 그라데이션 + 핫코랄/마젠타라서, 채팅은 더 차분한 무채색 베이스 + 단일 네온 액센트로 분리.

**운영 노트.**
- **모드레이션 시그널 컬러.** 랜덤·익명 채팅은 신고/차단/경고 UI가 자주 등장한다. brand color와 색역이 겹치지 않게 빨강/주황을 따로 확보 (`#EF4444` 류).
- **온라인/오프라인 status indicator.** 거의 항상 그린 = 온라인. brand Primary가 그린 계열이면 status는 시안 등 다른 색역으로 분리 (Line은 brand 그린이라서 status를 다르게 처리).
- **Unread / Active 배지.** brand accent와 같은 색역이 자연스럽다. 다만 너무 채도 높으면 사용자 피로(N+1 알림 트라우마)에 기여 — 톤 다운된 채도 권장.
- **랜덤 채팅 특수 신호.** "지금 활발한 방", "새 방 열림" 같은 상태 표시가 자주 등장. brand accent를 이런 용도로 쓰면 강조가 분명해진다.

**SNS/콘텐츠와의 분리.** 메신저는 "1:1 또는 소수와의 대화" 코드, SNS는 "1:N 콘텐츠 방송" 코드라서 강조점이 다르다. 메신저는 보라·시안·그린이 강하고, SNS는 빨강·검정·그라데이션이 강하다. 랜덤 채팅 룸은 메신저에 훨씬 가깝다.

## 미디어 / 콘텐츠 / SNS

**관습 색.** 정답이 없다. 빨강(유튜브/넷플릭스), 파랑(페이스북/트위터), 보라(인스타 그라데이션), 검정(틱톡), 노랑(스냅챗). 각자 정체성으로 차별화.
**원칙.** **하나의 강한 시그니처 컬러**가 SNS/콘텐츠 브랜드의 핵심. 5색 다 무난하면 기억이 안 남는다.

## 비영리 / 환경 / 공공

**관습 색.** 그린(환경), 블루(물·평화), 따뜻한 황톳빛(아프리카 NGO 등 지역 정체성).
**피해야 할 것.** 너무 럭셔리한 골드/검정 (비영리 정체성과 충돌).
**차별화 카드.** 모던 비영리는 흰 배경 + 한 가지 강한 컬러 + 사진. 깔끔하고 진중하다.

## 마지막 원칙

도메인 관습은 출발점이지 도착점이 아니다. 사용자가 명시적으로 "달라 보이고 싶다"고 하면 관습을 깨고 차별화 카드를 적극 권장. 명시적으로 안 했어도, 3개 팔레트 중 1개는 관습을 충실히 따르고, 1개는 살짝 비틀고, 1개는 과감히 비켜가는 식으로 펼쳐 보여주는 게 좋다 — 사용자가 어디까지 모험할 의사가 있는지 그 자리에서 가늠할 수 있게.
