# Domain Conventions

각 분야에는 사용자가 "이건 그 카테고리답다"고 무의식적으로 느끼는 색 코드가 있다. 무조건 따라야 하는 룰은 아니지만, 의식하지 않은 채로 어기면 카테고리가 안 읽힌다. 의식적으로 어기면 강한 차별화 무기가 된다.

## Fintech / 자산 / 결제

**관습 색.** 네이비, 딥 블루, 다크 그린. 이유: 은행 200년 역사 + 안정감/돈/자연의 무의식적 코드.
**참고 브랜드.** Chase·Citi(네이비), 토스(네이비→블루), 카카오뱅크(옐로 액센트는 메신저 DNA), 신한SOL(짙은 블루).
**자주 보는 신호색.** 상승(좋음): 짙은 그린 (#0F9D58 류). **한국 주식/암호화폐는 빨강=상승, 파랑=하락** (글로벌과 정반대).
**피해야 할 것.** 너무 채도 높은 핫핑크/마젠타/네온 — 아무리 MZ 타깃이라도 "내 돈"을 맡기는 맥락에선 가벼워 보인다. "모던하고 트렌디"가 사용자 요청에 있더라도 Primary는 신뢰 톤으로, Accent에서만 트렌드를 풀어주는 게 안전하다.
**차별화 카드 (강함→약함 순서).**
- **차콜 + 골드/앰버** — Bloomberg Terminal × Apple Dark 톤. "디지털 골드" 내러티브에 강력. **암호화폐/투자 자산** 분야에서 가장 영리한 차별화 카드. 비트코인 자체가 골드 메타포라서 즉시 읽힌다.
- **보라/바이올렛** — Revolut, Linear, Arc 계열. 똑똑하고 모던한 "테크 자산". MZ 타깃 핀테크에서 가장 흔한 차별화.
- **차콜 + 민트/그린 액센트** — Cash App, 토스증권의 다크모드. 청량하고 부담 없는 입문 톤.
- **따뜻한 베이지/우드** — 국내 자산관리 신생 브랜드(WealthX, 박곰핀 등). "따뜻한 자산 상담사" 톤.

**암호화폐/Web3 특수 사항.**
- 한국 거래소(업비트·빗썸): 호가창에서 **빨강=상승, 파랑=하락**. brand primary가 빨강/파랑 계열이면 신호색과 충돌 → 사용자 인지 부조화. **brand primary와 시그널 컬러는 절대 같은 색역으로 쓰지 마세요.** 예: Primary가 mint면 상승은 mint가 아니라 빨강(`#F04452`)으로 유지.
- 글로벌 사용자 비중이 크다면 **빨강=하락, 그린=상승** 옵션 토글을 제공하는 게 안전.
- "디지털 골드" 앵글이 강력 — 비트코인을 가치 저장 수단으로 포지셔닝하면 골드 톤이 즉시 의미를 전달한다.

**Trust vs Trendy 텐션 해소법.** 사용자 요청에 "신뢰감"과 "모던/트렌디"가 함께 들어있으면 **Primary는 신뢰, Accent는 트렌드** 분리가 가장 안정적. Primary를 가장 모험적으로 갈수록 사용자가 "이거 진짜 내 돈 맡길 수 있나"는 의구심이 커진다.

## Healthcare / Wellness / 명상

**관습 색.** 부드러운 블루, 민트, 세이지 그린, 베이지. 이유: 청결/회복/자연.
**참고 브랜드.** Calm(딥 블루→그라데이션), Headspace(주황·옐로 = 의도적 차별화), Whoop(블랙+그린 = 테크-웰니스), Oura(차콜+골드 = 프리미엄), 마보(따뜻한 베이지).
**피해야 할 것.** 너무 채도 높은 빨강(병원 응급 코드 연상), 형광 컬러, 새까만 다크모드(병자가 죽음을 연상). 검정 일색 헬스케어는 거의 없다. **종교·뉴에이지 코드(보라+골드 = 영적, 인디고+퍼플 = 신비주의)** 도 피하라는 요청이 많다.
**차별화 카드.**
- 따뜻한 살구색/피치 (불안 완화 톤, Headspace가 대표적)
- 라벤더/페일 퍼플 (수면 앱)
- 모스/세이지 그린 (자연 회복)
- 차콜 + 골드 (Oura 톤, 프리미엄 테크-웰니스)
- 따뜻한 베이지+테라코타 (디지털 디톡스, 인간적 톤)

**의료기기/임상** 쪽은 조금 더 차갑고 중성적인 톤 (시멘트 그레이, 차가운 블루)이 신뢰감을 준다.

## F&B / 배달 / 푸드

**관습 색.** 따뜻한 주황, 빨강, 옐로. 이유: 식욕 자극. 맥도날드/버거킹/우버이츠/요기요 다 따뜻한 색.
**참고 브랜드.** 배달의민족(민트→옐로), 쿠팡이츠(핑크), 마켓컬리(보라 = 의도적 차별화로 "프리미엄 새벽배송"), 스윗밸런스(헬시 그린+크림), 풀무원(딥 그린+자연색).
**피해야 할 것.** 파랑·보라 위주 (식욕 감소, 자연계에 거의 없는 음식 색). 단, **컬리처럼 의도적으로 프리미엄/큐레이션 차별화**를 노릴 땐 보라가 강력해진다.
**차별화 카드.**
- 헬시푸드/샐러드 → **그린+크림+우드** (Sweetgreen, 스윗밸런스, dimi)
- 프리미엄 다이닝 → **차콜+골드+와인** (와인 셀러 톤)
- 디저트/베이커리 → **파스텔 핑크/크림+초콜릿 브라운**
- 한식/토속 → **테라코타, 머스타드, 옻갈/감색, 한지 크림** (적당히 톤 다운된 흙·식물·토기 색)
- 프리미엄 큐레이션 → **딥 퍼플/와인+크림** (마켓컬리 패턴)

**한식 + 식단관리**는 보통 두 톤이 충돌한다 — 한식 토속(따뜻한 갈색)과 헬시(그린)는 색조가 가깝지 않아서 한 팔레트 안에서 둘 다 강하게 쓰면 어수선해진다. 한 팔레트 안에서는 한쪽을 기준으로 잡고 다른 쪽은 액센트나 보조로 빼는 게 깔끔하다.

## SaaS / B2B 협업 / 생산성

**관습 색.** 블루(노션 비포어, 슬랙 보라, 리니어 보라, 피그마 검정 등 사실 다양). 핵심은 "이성적이고 방해되지 않는" 톤.
**피해야 할 것.** 채도 너무 높은 풀컬러 — 작업하면서 8시간 보면 눈 아프다.
**차별화 카드.** 보라(노션/리니어 계열의 똑똑함), 그린(긍정/생산성), 다크모드 우선 디자인.
**B2B는 의외로 무난한 게 정답일 때가 많다.** 결정권자가 IT팀장이고 그 사람의 미감보다는 "사내에서 거부감 없는지"가 더 중요해서.

## 교육 / EdTech / 키즈

**관습 색.** 밝은 블루, 그린, 따뜻한 옐로. 이유: 신뢰 + 성장 + 활기.
**키즈**는 더 비비드하게 — 빨강/파랑/노랑 원색 + 일러스트. 단 학부모도 봐야 하니 너무 유치하면 거부감.
**차별화 카드.** 코딩 교육은 모노톤+컬러풀 액센트 (개발자스러운 분위기). 입시는 차분한 네이비/딥 그린 (진중함). 어학은 빨강/주황 (활동성).

## 게임 / 엔터테인먼트

**관습 색.** 다크 베이스 + 비비드 네온 액센트. 이유: OLED 화면 발광감, 몰입.
**차별화 카드.** 캐주얼 모바일 게임은 오히려 밝은 베이스 (애니팡, 캔디크러시). 코어 게이밍은 다크 + 보라/시안/레드 네온.

## Luxury / 패션 / 뷰티

**관습 색.** 블랙, 화이트, 골드, 카멜, 와인. 이유: 비싼 광고지·잡지 80년의 유산.
**피해야 할 것.** 광택 있는 비비드 컬러, 채도 높은 형광색.
**차별화 카드.** 화이트+소프트핑크(클린 뷰티), 차콜+세이지(미니멀 럭셔리), 풀 컬러풀 그라데이션(MZ 패션 브랜드).

## Mobility / 모빌리티 / 자동차

**관습 색.** 블랙, 다크 그레이, 다크 블루 (자동차 자체가 거의 다 무채색이라).
**차별화 카드.** EV/그린 모빌리티는 그린+화이트. 차량 공유는 따뜻한 톤(우버블랙 vs 우버X 의도된 분리).

## 미디어 / 콘텐츠 / SNS

**관습 색.** 정답이 없다. 빨강(유튜브/넷플릭스), 파랑(페이스북/트위터), 보라(인스타 그라데이션), 검정(틱톡), 노랑(스냅챗). 각자 정체성으로 차별화.
**원칙.** **하나의 강한 시그니처 컬러**가 SNS/콘텐츠 브랜드의 핵심. 5색 다 무난하면 기억이 안 남는다.

## 비영리 / 환경 / 공공

**관습 색.** 그린(환경), 블루(물·평화), 따뜻한 황톳빛(아프리카 NGO 등 지역 정체성).
**피해야 할 것.** 너무 럭셔리한 골드/검정 (비영리 정체성과 충돌).
**차별화 카드.** 모던 비영리는 흰 배경 + 한 가지 강한 컬러 + 사진. 깔끔하고 진중하다.

## 디포지티브: 무드 키워드 사전

| 무드 키워드 | 어울리는 톤 | 메모 |
|---|---|---|
| 신뢰감/안정감 | 네이비, 딥 블루, 차콜 | 채도 낮추기 |
| 친근함/활기 | 오렌지, 코럴, 옐로 | 채도 높이고 명도 살짝 |
| 럭셔리/프리미엄 | 블랙, 차콜, 골드, 와인 | 채도 낮고 명도 극단 |
| 미니멀/깔끔 | 오프화이트, 라이트 그레이, 한 가지 강한 컬러 | 컬러 가짓수 줄이기 |
| 모던/테크 | 보라, 시안, 짙은 그라데이션 | 채도 살짝 높고 디지털 느낌 |
| 따뜻함/자연 | 베이지, 테라코타, 세이지, 머스타드 | 톤 다운된 흙·식물 색 |
| 대담함/도전 | 네온, 비비드 빨강, 강한 보색 | 채도 끝까지 |
| 차분함/명상 | 라벤더, 페일 블루, 미스트 그린 | 채도 낮고 명도 높음 |
| 펀/캐주얼 | 비비드 옐로/핑크/시안 조합 | 보색 대비 |
| 차세대/미래 | 다크 + 네온 액센트 | 발광감 |

## 마지막 원칙

도메인 관습은 출발점이지 도착점이 아니다. 사용자가 명시적으로 "달라 보이고 싶다"고 하면 관습을 깨고 차별화 카드를 적극 권장. 명시적으로 안 했어도, 3개 팔레트 중 1개는 관습을 충실히 따르고, 1개는 살짝 비틀고, 1개는 과감히 비켜가는 식으로 펼쳐 보여주는 게 좋다 — 사용자가 어디까지 모험할 의사가 있는지 그 자리에서 가늠할 수 있게.
